# Aybist PHP Admin Paneli

Pure PHP 8.2 ile Yönetici Paneli

## Klasör Yapısı

- assets
  - css
  - img
  - js
  - fonts

- inc
  * config dosyaları

- libary
  * 3. taraf PHP modülleri

- pages
  * PHP alt sayfalar

- uplodas
  * sunucuya gönderilen medya dosyaları

# Kurulum

- $ composer install
- $ npm install

# Gereksinimler

- PHP 8.2
- Composer 2.2

# Proje Boyutu

- kütüphaneler dahil: 500kb
- kütüphaneler hariç: 30kb
